## Introduction

MSFS Aviation Unit Converter is a unit conversion tool designed specifically for aviation enthusiasts, supporting bidirectional conversion of various commonly used aviation units. This tool is particularly suitable for Microsoft Flight Simulator (MSFS) and other flight simulation players.

## How to Use

1. **Run the Program**
   - Download the exe file for your corresponding system
   - Double-click to run the program, no installation required

2. **Select Conversion Type**
   - Choose the type of unit conversion you need from the dropdown menu
   - Supported unit types:
     - Meters ‚Üî Feet
     - Hectopascals ‚Üî Inches of Mercury
     - Pounds ‚Üî Kilograms
     - Nautical Miles ‚Üî Kilometers
     - Knots ‚Üî Kilometers per hour

3. **Enter Values**
   - Enter a value in the left input box
   - The right side will automatically display the conversion result
   - You can also enter a value on the right side, and the left side will show the reverse conversion result

4. **Switch Language**
   - Click the language button in the upper right corner (‰∏≠Êñá/English)
   - The interface will immediately switch to the corresponding language

5. **Clear Inputs**
   - Click the "Clear" button to clear all input boxes

## Technical Information

This program is developed based on Python, using the customtkinter library to build the graphical user interface. It is compatible with Windows, macOS, and Linux systems (compilation required for systems other than Windows), providing a cross-platform, ready-to-use experience.
